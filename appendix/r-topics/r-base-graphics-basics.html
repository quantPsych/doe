Introduction

Hypothesis testing is a cornerstone of statistical analysis, used to evaluate assumptions about population parameters based on sample data. In R, several functions and packages provide extensive tools for hypothesis testing, ranging from basic tests in base R to more specialized tests available in external packages.

Overview

  • Packages/Functions:
    • Base R: t.test(), chisq.test(), var.test(), wilcox.test()
    • Additional Packages: stats::ks.test(), MASS::leveneTest()
  • Purpose:
    • To evaluate assumptions or hypotheses about data distributions and relationships.
  • Common Functions:
    • t.test() for t-tests.
    • chisq.test() for Chi-square tests.
    • ks.test() for Kolmogorov-Smirnov tests.
    • wilcox.test() for non-parametric Wilcoxon tests.

Use Cases

  • Comparing means between two groups.
  • Assessing the goodness-of-fit for categorical data.
  • Testing for equal variances across groups.
  • Non-parametric comparisons for skewed or ordinal data.

Benefits

  • Easy to implement with a range of built-in functions.
  • Highly extensible for complex scenarios.
  • Seamless integration with visualization and other analytical tools.

Getting Started

Loading the Package

Base R provides most hypothesis testing tools. Load the MASS package for additional tests.

library(MASS)

Basic Usage

Example: One-Sample T-test

Testing whether the mean of a sample is equal to a specific value.

set.seed(42)
sample_data <- rnorm(30, mean = 5, sd = 2)

t.test(sample_data, mu = 5)

    One Sample t-test

data:  sample_data
t = 0.29933, df = 29, p-value = 0.7668
alternative hypothesis: true mean is not equal to 5
95 percent confidence interval:
 4.199903 6.074444
sample estimates:
mean of x 
 5.137174

Key Functions and Features

t.test()

Description

Performs one-sample, two-sample, and paired t-tests to compare means.

Syntax

t.test(x, y = NULL, alternative = c("two.sided", "less", "greater"),
       mu = 0, paired = FALSE, var.equal = FALSE, conf.level = 0.95)
  • x, y: Numeric vectors of data values.
  • alternative: Type of hypothesis (“two.sided”, “greater”, “less”).
  • mu: Hypothesized mean (default is 0).
  • paired: Whether the test is paired (default is FALSE).
  • var.equal: Assume equal variances (default is FALSE).

Example: Two-Sample T-test

group1 <- rnorm(20, mean = 5)
group2 <- rnorm(20, mean = 6)

t.test(group1, group2, alternative = "two.sided", var.equal = TRUE)

    Two Sample t-test

data:  group1 and group2
t = -4.6894, df = 38, p-value = 3.486e-05
alternative hypothesis: true difference in means is not equal to 0
95 percent confidence interval:
 -2.1057327 -0.8358574
sample estimates:
mean of x mean of y 
 4.807940  6.278735

chisq.test()

Description

Tests for independence or goodness-of-fit in categorical data.

Syntax

chisq.test(x, y = NULL, correct = TRUE, simulate.p.value = FALSE, rescale.p = FALSE)

Example: Independence Test

observed <- matrix(c(30, 20, 50, 40), ncol = 2)

chisq.test(observed)

    Pearson's Chi-squared test with Yates' continuity correction

data:  observed
X-squared = 0.10954, df = 1, p-value = 0.7407

wilcox.test()

Description

Performs the Wilcoxon rank-sum test (non-parametric alternative to the t-test).

Syntax

wilcox.test(x, y = NULL, alternative = c("two.sided", "less", "greater"), mu = 0, paired = FALSE, exact = NULL)

Example: Wilcoxon Test

x <- rnorm(30)
y <- rnorm(30, mean = 1)

wilcox.test(x, y, alternative = "less")

    Wilcoxon rank sum exact test

data:  x and y
W = 271, p-value = 0.003824
alternative hypothesis: true location shift is less than 0

In-Depth Examples

Comparing Means Across Groups

Title: Paired T-test for Preand Post-Treatment Data

Objective

Evaluate the effect of a treatment by comparing preand post-treatment scores.

Steps

  1. Prepare the data.
  2. Apply the paired t-test.
  3. Visualize the differences.
pre <- c(12, 14, 11, 10, 13, 15)
post <- c(14, 16, 12, 11, 15, 17)

t.test(pre, post, paired = TRUE)

    Paired t-test

data:  pre and post
t = -7.9057, df = 5, p-value = 0.0005211
alternative hypothesis: true mean difference is not equal to 0
95 percent confidence interval:
 -2.208593 -1.124740
sample estimates:
mean difference 
      -1.666667

Output

Interpret the p-value to determine if there is a significant difference between preand post-treatment scores.

Goodness-of-Fit Test

Title: Testing Dice Fairness

Objective

Check if a six-sided die is fair.

Steps

  1. Collect observed frequencies of each side.
  2. Specify expected frequencies.
  3. Apply the Chi-square goodness-of-fit test.
observed <- c(12, 15, 14, 13, 16, 10)
expected <- rep(1 / 6, 6) * sum(observed)

chisq.test(observed, p = expected / sum(expected))

    Chi-squared test for given probabilities

data:  observed
X-squared = 1.75, df = 5, p-value = 0.8825

Output

Evaluate the p-value to determine if the die is fair.

Troubleshooting and FAQs

Common Issues

  1. Error: Data must be numeric
    • Ensure data is numeric by using as.numeric().
  2. Warning: p-value approximation
    • Increase the sample size or use simulate.p.value = TRUE.

FAQs

  1. What is the minimum sample size for a t-test?
    • At least 5 observations per group are recommended.
  2. Can I test for normality before t-tests?
    • Use shapiro.test() to check normality.

Best Practices

  • Always visualize your data before performing tests.
  • Check assumptions (e.g., normality for t-tests, independence for Chi-square tests).
  • Use appropriate alternatives (e.g., Wilcoxon test for non-normal data).

Conclusion

Hypothesis testing in R is a versatile and powerful tool for statistical analysis. From basic t-tests to advanced non-parametric methods, R provides comprehensive support for validating assumptions and making data-driven decisions.

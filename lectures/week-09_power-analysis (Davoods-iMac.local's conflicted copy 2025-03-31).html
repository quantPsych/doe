1. Power Analysis for One-Way ANOVA

What is One-Way ANOVA?

One-way ANOVA tests whether the means of three or more groups differ significantly. For example, imagine comparing the average test scores of students taught using three teaching methods (Method A, Method B, Method C).

Key Concepts in Power Analysis

  • Power (1 - β): The probability of correctly rejecting a false null hypothesis (typically targeted at 0.80 or higher).
  • Significance Level (α): The probability of a Type I error (rejecting a true null hypothesis), often set at 0.05.
  • Effect Size: A measure of the magnitude of differences between group means, quantified in ANOVA using Cohen’s f.
  • Sample Size (r): The number of observations per group, which we often solve for in power analysis.
  • Noncentrality Parameter (λ): A parameter that shifts the F-distribution under the alternative hypothesis, used to compute power.

Effect Size: Cohen’s F

Cohen’s f measures the standardized difference between group means relative to within-group variability. The formula is:

\[ f = \sqrt{\frac{\sum_{i=1}^k (\mu_i - \mu)^2 / k}{\sigma^2}} \]

Where:

  • \(k\): Number of groups– In the textbook, \(\nu\) is used instead of \(k\).
  • \(\mu_i\): Mean of group \(i\).
  • \(\mu\): Overall mean across all groups.
  • \(\sigma^2\): Within-group variance (assumed equal across groups).

Interpretation (Cohen’s benchmarks):

  • Small: \(f = 0.1\)
  • Medium: \(f = 0.25\)
  • Large: \(f = 0.4\)

Example Calculation of Cohen’s f

Suppose we have three groups (\(k = 3\)) with means \(\mu_1 = 10\), \(\mu_2 = 12\), \(\mu_3 = 14\), and within-group variance \(\sigma^2 = 16\). The overall mean is:

\[ \mu = \frac{10 + 12 + 14}{3} = 12 \]

Numerator of \(f^2\):

\[ \frac{\sum (\mu_i - \mu)^2}{k} = \frac{(10 - 12)^2 + (12 - 12)^2 + (14 - 12)^2}{3} = \frac{4 + 0 + 4}{3} = \frac{8}{3} \approx 2.667 \]

\[ f = \sqrt{\frac{2.667}{16}} = \sqrt{0.1667} \approx 0.408 \]

This is a large effect size, close to Cohen’s \(f = 0.4\).

Noncentrality Parameter (λ)

The noncentrality parameter quantifies the shift in the F-distribution under the alternative hypothesis:

\[ \lambda = \frac{r \sum_{i=1}^k (\mu_i - \mu)^2}{\sigma^2} \]

Where \(r\) is the sample size per group (assuming balanced design). Using Cohen’s f:

\[ \lambda = r k f^2 \]

Example

Using the above means and \(\sigma^2 = 16\), with \(r = 10\) per group:

\[ \sum (\mu_i - \mu)^2 = 4 + 0 + 4 = 8 \]

\[ \lambda = \frac{10 \times 8}{16} = 5 \]

Or via \(f\):

\[ \lambda = 10 \times 3 \times (0.408)^2 = 30 \times 0.1665 \approx 5 \]

Power Calculation

Power is calculated using the noncentral F-distribution:

  • Numerator degrees of freedom: \(df_1 = k - 1\)
  • Denominator degrees of freedom: \(df_2 = N - k\), where \(N = r k\)
  • Critical F-value: \(F_{\text{crit}}\) at \(\alpha\), found from the F-distribution.
  • Power: \(1 - P(F < F_{\text{crit}} | \lambda)\), where \(F\) follows a noncentral F-distribution.

Numerical Example

Goal: Determine the sample size per group (\(r\)) for \(k = 3\) groups, medium effect size (\(f = 0.25\)), \(\alpha = 0.05\), and power = 0.80.

Using R’s pwr Package

library(pwr)

k <- 3
f <- 0.25
alpha <- 0.05
power <- 0.80

result <- pwr.anova.test(k = k, f = f, sig.level = alpha, power = power)
print(result)

     Balanced one-way analysis of variance power calculation 

              k = 3
              n = 52.3966
              f = 0.25
      sig.level = 0.05
          power = 0.8

NOTE: n is number in each group

Output:

     Balanced one-way analysis of variance power calculation 

              k = 3
              n = 52.39771
              f = 0.25
      sig.level = 0.05
          power = 0.8

NOTE: n is number in each group

We need approximately \(r = 53\) subjects per group, so total \(N = 3 \times 53 = 159\).

Manual Verification

  1. Degrees of Freedom:
    • \(df_1 = k - 1 = 2\)
    • \(df_2 = N - k = 3 \times 53 - 3 = 156\)
  2. Noncentrality Parameter:

\[ \lambda = r k f^2 = 53 \times 3 \times 0.25^2 = 159 \times 0.0625 = 9.9375 \]

  1. Critical F-value:

\[ F_{\text{crit}} = qf(0.95, 2, 156) \approx 3.05 \]

  1. Power:

\[ \text{Power} = 1 - pf(3.05, 2, 156, ncp = 9.9375) \]

power_manual <- 1 - pf(qf(0.95, 2, 156), 2, 156, ncp = 9.9375)
print(power_manual)
[1] 0.8048873

Output: Approximately 0.803, confirming the power is close to 0.80, validating the pwr result.

2. Power Analysis for Two-Way ANOVA

What is Two-Way ANOVA?

Two-way ANOVA examines the effects of two categorical factors (e.g., teaching method and study time) on a response variable, testing:

  • Main Effect of Factor A
  • Main Effect of Factor B
  • Interaction Effect (A × B)

Key Concepts

Power analysis in two-way ANOVA requires separate calculations for each effect, adjusting effect sizes and degrees of freedom accordingly.

Effect Size: Cohen’s F for Two-Way ANOVA

For each effect, Cohen’s f is the ratio of the effect’s standard deviation to the within-group standard deviation (\(\sigma\)).

  • Main Effect A:

\[ f_A = \frac{\sigma_A}{\sigma}, \quad \sigma_A = \sqrt{\frac{\sum_{i=1}^a (\alpha_i)^2}{a}} \]

where \(\alpha_i = \mu_{i.} - \mu\) (deviation of Factor A’s level means from the grand mean).

  • Main Effect B:

\[ f_B = \frac{\sigma_B}{\sigma}, \quad \sigma_B = \sqrt{\frac{\sum_{j=1}^b (\beta_j)^2}{b}} \]

where \(\beta_j = \mu_{.j} - \mu\).

  • Interaction Effect:

\[ f_{AB} = \frac{\sigma_{AB}}{\sigma}, \quad \sigma_{AB} = \sqrt{\frac{\sum_{i=1}^a \sum_{j=1}^b (\gamma_{ij})^2}{a b}} \]

where \(\gamma_{ij} = \mu_{ij} - \mu_{i.} - \mu_{.j} + \mu\) (interaction terms).

Noncentrality Parameter (λ)

Assuming \(r\) replicates per cell, \(a\) levels of Factor A, and \(b\) levels of Factor B:

  • Main Effect A:

\[ \lambda_A = \frac{r b \sum_{i=1}^a (\alpha_i)^2}{\sigma^2} = r b a f_A^2 \]

  • Main Effect B:

\[ \lambda_B = \frac{r a \sum_{j=1}^b (\beta_j)^2}{\sigma^2} = r a b f_B^2 \]

  • Interaction:

\[ \lambda_{AB} = \frac{r \sum_{i=1}^a \sum_{j=1}^b (\gamma_{ij})^2}{\sigma^2} = r a b f_{AB}^2 \]

Degrees of Freedom

  • Total sample size: \(N = a b r\)
  • Main Effect A: \(df_1 = a - 1\), \(df_2 = N - a b\)
  • Main Effect B: \(df_1 = b - 1\), \(df_2 = N - a b\)
  • Interaction: \(df_1 = (a - 1)(b - 1)\), \(df_2 = N - a b\)

Numerical Example

Scenario: A 2×2 design (Factor A: 2 levels, Factor B: 2 levels). We want to detect a main effect of Factor A with a difference of 4 units between levels, \(\sigma^2 = 7.5\), power = 0.80, \(\alpha = 0.05\). Find \(r\) (replicates per cell).

Step 1: Define the Effect

Suppose Factor A’s marginal means are \(\mu_{1.} = 12\), \(\mu_{2.} = 8\), so \(\mu_{1.} - \mu_{2.} = 4\). Grand mean \(\mu = 10\). Effects are:

\[ \alpha_1 = 12 - 10 = 2, \quad \alpha_2 = 8 - 10 = -2 \]

\[ \sum \alpha_i^2 = 2^2 + (-2)^2 = 8 \]

Step 2: Noncentrality Parameter

\[ \lambda_A = \frac{r b \sum \alpha_i^2}{\sigma^2} = \frac{r \times 2 \times 8}{7.5} = \frac{16 r}{7.5} \]

Step 3: Degrees of Freedom

  • \(df_1 = a - 1 = 1\)
  • \(df_2 = N - a b = 2 \times 2 \times r - 4 = 4r - 4\)

Step 4: Manual Power Calculation

For \(r = 5\):

  • \(N = 2 \times 2 \times 5 = 20\)
  • \(df_2 = 20 - 4 = 16\)
  • \(\lambda_A = \frac{16 \times 5}{7.5} \approx 10.67\)
  • \(F_{\text{crit}} = qf(0.95, 1, 16) \approx 4.49\)

\[ \text{Power} = 1 - pf(4.49, 1, 16, ncp = 10.67) \]

power <- 1 - pf(qf(0.95, 1, 16), 1, 16, ncp = 10.67)
print(power)
[1] 0.8654718

Output: ~0.85 > 0.80.

For \(r = 4\):

  • \(df_2 = 4 \times 4 - 4 = 12\)
  • \(\lambda_A = \frac{16 \times 4}{7.5} \approx 8.53\)
  • \(F_{\text{crit}} \approx 4.75\)
  • Power ~0.71 <- 0.80.

Thus, \(r = 5\) is the smallest integer satisfying power ≥ 0.80.

Using R (Approximation via pwr)

Since pwr doesn’t directly handle two-way ANOVA, we approximate the main effect as a t-test (since \(df_1 = 1\)):

\[ d = \frac{\mu_{1.} - \mu_{2.}}{\sqrt{\sigma^2 / (r b)}} = \frac{4}{\sqrt{7.5 / (r \times 2)}} \]

But for precision, use manual F-test or specialized tools like G*Power. The pwr2 package isn’t standard for this; we rely on pwr for one-way or custom calculations.

Summary

  • One-Way ANOVA: Use \(k\), \(f\), \(\alpha\), and power to find \(r\) per group (e.g., \(r = 53\) for \(k = 3\), \(f = 0.25\)). Verify with pwr.anova.test.
  • Two-Way ANOVA: Calculate power for each effect separately, defining \(f\) and \(\lambda\) per effect. For a 2×2 main effect with a difference of 4, \(r = 5\) per cell suffices.

These methods ensure your experiment is well-powered to detect meaningful differences!

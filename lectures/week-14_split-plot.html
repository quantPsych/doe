1. Introduction

Split-plot designs are a powerful class of experimental designs that originated in agricultural settings but now see extensive use across engineering and scientific fields where certain factors are hard to change and others are easy to change. First formalized by R. A. Fisher at Rothamsted Experimental Station, these designs address practical limitations in experimentation—for instance, when adjusting equipment temperature is costly or time-consuming (Kempthorne, 1977; Scheffé, 1959).

1.1 Historical Context

  • Early Development: Split-plot methodology emerged from agricultural research, where entire fields (whole plots) were assigned to certain treatments (e.g., irrigation methods), and sub-portions of each field (split plots) received additional treatments (e.g., fertilizer types).
  • Modern Applications: Today, split-plot designs are prevalent in industrial experiments (Dean et al., 2017), such as semiconductor manufacturing or paint-curing processes. They enable investigators to manipulate “hard-to-change” variables at the whole-plot level while fine-tuning other variables at the split-plot level.

1.2 Learning Objectives

After working through this material, you will be able to:

  1. Recognize when split-plot designs are appropriate.
  2. Formulate the statistical models underpinning split-plot experiments.
  3. Analyze split-plot data using R and interpret the output.
  4. Understand the hierarchical (two-error) structure inherent in split-plot designs.
  5. Conduct diagnostic checks and manage potential pitfalls such as unbalanced data or pseudo-replication.

2. Fundamental Concepts

2.1 Two-Error Structure

A defining feature of split-plot designs is the presence of two distinct experimental error terms (Dean et al., 2017; Montgomery & Runger, 2010; Christensen, 2018):

  1. Whole-Plot Error (\(\sigma^2_W\))
    • Arises from variation among the “hard-to-change” factors (e.g., temperature, irrigation).
    • Typically larger due to the cost and complexity of changing these factors.
    • Example: Oven-to-oven variability when testing different oven temperatures.
  2. Split-Plot Error (\(\sigma^2_S\))
    • Captures variability within each whole plot for the “easy-to-change” factors.
    • Usually smaller since these factors are more straightforward to manipulate.
    • Example: Variation among parts painted in the same oven run.

This nested structure means that each whole plot is subdivided into several split plots, and different analysis methods are needed to properly separate the two sources of variability.

2.2 Intuitive Analogy: Paint Curing Process

Consider an industrial paint-curing process:

  • Whole-Plot Factor: Oven Temperature (Low, Medium, High)
  • Split-Plot Factor: Paint Type (A, B, C)
  • Response: Coating Durability

Running the experiment:

  1. Set the oven to a specific temperature (hard to change).
  2. Paint several parts (easy to change) in that oven run.
  3. Measure the durability of each part’s coating.

Because the oven temperature remains fixed for a batch of painted parts, any variation among parts at the same temperature is captured by \(\sigma^2_S\) (split-plot error), whereas variation across different oven temperatures is captured by \(\sigma^2_W\) (whole-plot error).

3. Statistical Model

Adopting the notation from Montgomery & Runger (2010) and Christensen (2018), a balanced split-plot model might look like this:

\[ Y_{hiujt} = \mu + \theta_h + \alpha_i + \epsilon_{Wiu(h)} + \beta_j + (\alpha\beta)*{ij} + \epsilon*{Sjt(hiu)}, \]

where:

  • \(\mu\) is the overall mean.
  • \(\theta_h\) is the block (or replicate) effect.
  • \(\alpha_i\) is the main effect of the whole-plot factor (e.g., temperature).
  • \(\beta_j\) is the main effect of the split-plot factor (e.g., paint type).
  • \((\alpha\beta)_{ij}\) is the interaction effect of the wholeand split-plot factors.
  • \(\epsilon_{Wiu(h)} \sim N(0, \sigma_W^2)\) represents the whole-plot error.
  • \(\epsilon_{Sjt(hiu)} \sim N(0, \sigma_S^2)\) represents the split-plot error.

4. Analysis of Variance (ANOVA)

Because of the nested structure, the ANOVA table partitions variability into different sources. A typical layout for a two-factor split-plot design (Christensen, 2018) is:

Source df EMS
Blocks \(b-1\) \(\sigma^2_S + s\sigma^2_W + as\sigma^2_B\)
Whole-Plot Factor (A) \(a-1\) \(\sigma^2_S + s\sigma^2_W + bs\sigma^2_A\)
Whole-Plot Error \((b-1)(a-1)\) \(\sigma^2_S + s\sigma^2_W\)
Split-Plot Factor (B) \(s-1\) \(\sigma^2_S + br\sigma^2_B\)
A×B Interaction \((a-1)(s-1)\) \(\sigma^2_S + r\sigma^2_{AB}\)
Split-Plot Error \(a(b-1)(s-1)\) \(\sigma^2_S\)

Where \(b\) is the number of blocks (or replicates), \(a\) is the number of whole-plot factor levels, and \(s\) is the number of split-plot factor levels.

5. Implementation in R

Below is a basic workflow demonstrating how to analyze a split-plot design using lme4 and emmeans.

library(lme4)
library(emmeans)

set.seed(123)

# Example Data: 
# Whole-plot factor = temp (Low, Medium, High)
# Split-plot factor = speed (Slow, Fast)
# Blocks = 4
data_split <- data.frame(
  block = rep(1:4, each = 6),
  temp = rep(rep(c("Low", "Med", "High"), each = 2), 4),
  speed = rep(c("Slow", "Fast"), times = 12),
  yield = rnorm(24, mean = 100, sd = 10)
)

# Fit the mixed-effects model (split-plot structure)
model_split <- lmer(yield ~ temp * speed + (1|block) + (1|block:temp),
                    data = data_split)

# ANOVA
anova(model_split, type = "III")

7. Advanced Topics

7.1 Power Analysis

In split-plot designs, we have distinct power calculations for the whole-plot factor and the split-plot factor (Dean et al., 2017). Each factor is tested against a different error term.

\[ \text{Power}^*_{\text{whole-plot}} \quad=\quad P\bigl(F^*_{a-1,(b-1)(a-1)} > F_{\alpha;a-1,(b-1)(a-1)} \;\bigm|\; \delta\bigr), \]

\[ \text{Power}^*_{\text{split-plot}} \quad=\quad P\bigl(F^*_{s-1,a(b-1)(s-1)} > F_{\alpha;s-1,a(b-1)(s-1)} \;\bigm|\; \delta\bigr). \]

  • \(\delta\) represents the noncentrality parameter linked to the effect size.
  • Software such as pwr, simr, or custom code can be used to approximate power.

7.2 Optimal Design Considerations

For large-scale experiments, cost constraints and logistical feasibility often dictate the chosen design structure (Kempthorne, 1977; Dean et al., 2017). Key elements include:

  1. Balance: Achieving uniform replication across factor levels.
  2. Resource Allocation: The ratio of whole-plot replicates to split-plot replicates.
  3. Variance Component Estimates: Anticipating or pilot-testing \(\sigma_W^2\) and \(\sigma_S^2\) to inform design decisions.

8. Common Pitfalls

8.1 Pseudo-Replication

A critical error is to treat measurements within a single whole plot as fully independent replicates for the whole-plot factor. This leads to inflated Type I error rates and overly optimistic standard errors.

Incorrect:

# Failing to account for hierarchical structure
lm_model <- lm(yield ~ temp * flow, data = semi_data)

Correct:

# Using a mixed-effects model to capture nested variability
lmer_model <- lmer(uniformity ~ temp * flow + (1|run) + (1|run:temp),
                   data = semi_data)

8.2 Unbalanced Data

Real-world settings often produce unbalanced data (missing cells, unequal replicates):

  • Use lmerTest for Satterthwaite or Kenward-Roger df approximations.
  • Examine residuals and leverage robust inference methods if necessary.
library(lmerTest)
unbalanced_data <- subset(semi_data, !(run == 4 & flow == "10")) # remove some observations
model_unbalanced <- lmer(
  uniformity ~ temp * flow +
    (1 | run) + (1 | run:temp),
  data = unbalanced_data
)
anova(model_unbalanced, ddf = "Satterthwaite")

9.2 Challenging Problems

  1. Deriving Expected Mean Squares
    • Task: Show that \(E(MS_{whole}) = \sigma^2_S + s\sigma^2_W\).
    • Hint: Partition the sum of squares for the whole plot and use properties of the normal distribution.
    • Reference: See derivations in Scheffé (1959) and Appendix A below.
  2. Power Calculation
    • Suppose \(\sigma_W^2 = 15\) and \(\sigma_S^2 = 5\). You have 3 levels of a whole-plot factor (A) and 4 levels of a split-plot factor (B), with 3 replicates (blocks).
    • Task: Outline how you would simulate data in R to estimate the power to detect a particular effect size \(\delta\) on factor A.
  3. Design Optimization
    • You have a cost function \(C = c_w n_w + c_s n_s\), and you want to keep the variance of \(\hat{\tau}*i - \hat{\tau}*{i'}\) below a threshold \(V_0\). Show how to derive the ratio \(n_w^* / n_s^*\) that minimizes cost subject to the variance constraint (Kempthorne, 1977).

Appendix A: Mathematical Derivations

A.1 Expected Mean Squares

Using the notation in Dean et al. (2017) and Scheffé (1959):

\[ \text{For a balanced split-plot design:} \]

  1. Whole-plot error:

\[ E(MS_{\text{whole}}) = \sigma^2_S + s\,\sigma^2_W. \]

  1. Split-plot error:

\[ E(MS_{\text{split}}) = \sigma^2_S. \]

The derivation partitions sums of squares and leverages properties of normal distributions (see Christensen, 2018, for a modern ANOVA treatment).

A.2 Power Analysis for Whole-Plot Factor

\[ \lambda \;=\; \frac{n_w \sum_{i=1}^a \tau_i^2}{a\,\sigma_W^2}, \]

where \(n_w\) is the number of whole-plot replicates, \(\tau_i\) are the treatment contrasts, and \(\sigma_W^2\) is the whole-plot variance. The power to detect a nonzero \(\tau_i\) at significance level \(\alpha\) becomes:

\[ 1 - \beta \;=\; P\!\bigl(F_{a-1,(b-1)(a-1)} > F_{\alpha;a-1,(b-1)(a-1)} \;\bigm|\; \lambda\bigr). \]

A.3 Optimal Design Theory

Following Kempthorne (1977):

\[ \text{Minimize } C = c_w n_w + c_s n_s \quad \text{subject to } Var(\hat{\tau}^*_i - \hat{\tau}^*_{i'}) \leq V_0. \]

If \(\sigma_W^2\) and \(\sigma_S^2\) are known (or estimated from pilot data):

\[ n_w^* = \sqrt{\frac{c_s}{c_w}} \;\frac{\sigma_W}{\sigma_S}\;n_s^*. \]

This balance ensures cost efficiency while meeting variance constraints.

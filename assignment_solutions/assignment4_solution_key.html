(a) Check the Equal-Variance Assumption

  1. Overview of four groups and sample sizes.
  2. Descriptive check via the ratio of sample SDs (≈ 2.03) suggests potential heteroscedasticity.
  3. Formal tests such as Bartlett’s test indicate unequal variances.
  4. A log transform is recommended—as confirmed by a Box-Cox analysis.

(d) Preferred Analysis

  • Log-Transformed Analysis is preferred when residual diagnostics confirm stabilization of variance.
  • Untransformed Analysis with Satterthwaite adjustment is less straightforward and used if transformation is unsatisfactory.

(c) Untransformed Data: Weighted Regression (Satterthwaite Approx)

  • Repeat part (b) using the untransformed data and Satterthwaite’s approximation for unequal variances.
  1. Formulate the quadratic contrast:
    • Determine quadratic contrast coefficients \(c_1, c_2, c_3, c_4\) that capture curvature (for example, these might be derived from the squared deviations from the mean of the dose levels, adjusted to sum to zero).
  2. Estimate the contrast:
    • Compute the contrast \(\hat{L} = \sum_{i=1}^4 c_i \bar{Y}_i\).
  3. Estimate the variance:
    • Calculate \(\widehat{\operatorname{Var}}(\hat{L}) = \sum_{i=1}^4 c_i^2 \frac{s_i^2}{n_i}\).
  4. Adjust degrees of freedom:
    • Use Satterthwaite’s formula:

\[ \nu = \frac{\left( \sum_{i=1}^4 c_i^2 \frac{s_i^2}{n_i} \right)^2}{\sum_{i=1}^4 \frac{\left(c_i^2 \frac{s_i^2}{n_i}\right)^2}{n_i-1}}, \]

to obtain the effective degrees of freedom.

  1. Perform the test:
    • Calculate the t-statistic:

\[ t = \frac{\hat{L}}{\sqrt{\widehat{\operatorname{Var}}(\hat{L})}}, \]

and compare this against the t-distribution with \(\nu\) degrees of freedom. A significant t-value would indicate a quadratic trend.

cont_mat <- contr.poly(5)
# Linear contrast coefficients
c_l <- cont_mat[, 1]
# Quadratic contrast coefficients
c_q <- cont_mat[, 2]

# Linear contrast
l_hat <- sum(c_l * tapply(speed_df$Crank, speed_df$Speed, mean))
var_l_hat <- sum(c_l^2 * tapply(speed_df$Crank, speed_df$Speed, var) / 3)
nu_l <- sum(
    (c_l^2 * tapply(speed_df$Crank, speed_df$Speed, var) / 3)^2 /
        (c_l^2 * tapply(speed_df$Crank, speed_df$Speed, var) / 3)^2 /
        2
)

t_l <- l_hat / sqrt(var_l_hat)
p_l <- 2 * pt(-abs(t_l), df = nu_l)

# Quadratic contrast
q_hat <- sum(c_q * tapply(speed_df$Crank, speed_df$Speed, mean))
var_q_hat <- sum(c_q^2 * tapply(speed_df$Crank, speed_df$Speed, var) / 3)
nu_q <- sum(
    (c_q^2 * tapply(speed_df$Crank, speed_df$Speed, var) / 3)^2 /
        (c_q^2 * tapply(speed_df$Crank, speed_df$Speed, var) / 3)^2 /
        2
)
t_q <- q_hat / sqrt(var_q_hat)
p_q <- 2 * pt(-abs(t_q), df = nu_q)

# Create a table for the contrasts, t-values, and p-values

contrast_table <- data.frame(
    Contrast = c("Linear", "Quadratic"),
    Est = c(l_hat, q_hat),
    se = sqrt(c(var_l_hat, var_q_hat)),
    df = c(nu_l, nu_q),
    t = c(t_l, t_q),
    p_valu = c(p_l, p_q)
)

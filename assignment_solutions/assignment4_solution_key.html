    Exercise 3: Margarine Experiment

    (a) Check Equal-Variance Assumption

    • Step 1: Plot the residuals or calculate variances for each group (Brands 1–3 for margarine and Brand 4 for butter).

    • Step 2: Use Bartlett’s test or Levene’s test to check for equality of variances:

        bartlett.test(MeltingTime ~ Brand, data = data)
      • If \(p\)-value is small (e.g., \(< 0.05\)), the equal-variance assumption is violated.

    • Step 3: If violated, apply a transformation: y′=yλy’ = y^Common transformations:

      • \(\lambda = 0\): \(\log(y)\)
      • \(\lambda = 0.5\): \(\sqrt{y}\)
      • \(\lambda = -1\): \(1/y\)

    Exercise 6: Bicycle Experiment

    (a) Evaluate Error Variances

    • Step 1: Plot standardized residuals against fitted values:

        plot(fitted(model), rstandard(model))

    • Step 2: Compute sample variances for each speed group and compare:

        tapply(CrankRates, Speed, var)

    (b) Transformation and Hypothesis Testing

    1. Apply a transformation: y′=yλy’ = y^Test linear and quadratic trends using ANOVA:

      model <- lm(TransformedCrankRates ~ poly(Speed, 2), data = data)
anova(model)
      • Null hypothesis: Linear or quadratic trends are negligible.
      • Use \(\alpha = 0.01\) for significance.

    (c) Hypothesis Testing with Untransformed Data

    1. Use Satterthwaite’s approximation for unequal variances:

      oneway.test(CrankRates ~ Speed, data = data, var.equal = FALSE)

    (d) Compare Methods

    • Transformation improves normality and stabilizes variances, leading to more reliable results.
    • Untransformed data analysis is simpler but less accurate if variances are unequal.
